5개 차원으로 학술 프레젠테이션의 완성도를 평가한다.
자동 채점 가능 항목은 `scripts/quality_scorer.py`로 산출, 나머지는 수동 평가.

## 총점 계산

```
총점 = Σ(차원별 점수 × 가중치) / Σ(가중치)
```

| 차원 | 가중치 | 자동 채점 |
|------|--------|----------|
| 학술 근거 (Academic Evidence) | 25% | 부분 (DOI 유효성만) |
| 콘텐츠 밀도 (Content Density) | 20% | 가능 |
| 시각적 균형 (Visual Balance) | 20% | 가능 |
| 논리적 흐름 (Logical Flow) | 20% | 불가 |
| 형식 일관성 (Format Consistency) | 15% | 가능 |

---

## 1. 학술 근거 (Academic Evidence) — 25%

핵심 주장이 검증 가능한 학술 근거로 뒷받침되는가.

| 점수 | 기준 |
|------|------|
| 5 | 모든 핵심 주장에 DOI 레퍼런스, 수치 정확, 가이드라인 판본 명시 |
| 4 | 대부분 레퍼런스 있음, 일부 수치 출처 누락 |
| 3 | 주요 주장에 레퍼런스 있으나 세부 수치 미검증 |
| 2 | 레퍼런스 산발적, 핵심 주장 근거 부족 |
| 1 | 레퍼런스 거의 없음 |

**채점 예시:**

5점:
```
LLM 기반 TNM 판정은 94.2% 정확도로 기존 rule-based (81.7%)
대비 유의한 향상 (p<0.001). [Kim et al., Nature Medicine 2024]
→ DOI 있음, 수치 정확, p-value 포함
```

2점:
```
LLM 기반 TNM 판정이 높은 정확도를 보였다.
→ 구체적 수치 없음, 레퍼런스 없음, "높은"은 주관적
```

**체크리스트:**
- [ ] 모든 수치에 출처가 있는가?
- [ ] DOI가 유효한가? (`scripts/doi_lookup.py --validate`로 검증)
- [ ] 가이드라인 참조에 판본이 명시되어 있는가? (예: AJCC 8th Edition)
- [ ] 비교 수치의 기준이 명확한가? (무엇 대비 향상인지)
- [ ] References 슬라이드가 마지막에 포함되어 있는가?

## 2. 콘텐츠 밀도 (Content Density) — 20%

슬라이드당 정보량이 적절한가. 과소(빈 슬라이드)도, 과밀(텍스트 벽)도 문제.

| 점수 | 기준 |
|------|------|
| 5 | 슬라이드당 핵심 메시지 1개, 텍스트 100-500자, 여백 충분 |
| 4 | 대부분 적절, 1-2장 밀도 불균형 |
| 3 | 일부 슬라이드 과밀 또는 과소 |
| 2 | 다수 슬라이드가 텍스트 벽이거나 거의 비어있음 |
| 1 | 전반적으로 밀도 부적절 |

**자동 채점 기준 (`quality_scorer.py`):**
- 텍스트 < 50자: 과소 → 2점
- 텍스트 > 2000자: 과밀 → 3점
- 50~2000자: 적절 → 5점
- shape 수 < 5: 요소 부족 (경고)
- shape 수 > 60: 요소 과다 (경고)

**채점 예시:**

5점: 슬라이드에 제목(1줄) + 핵심 포인트(3-4줄) + 차트 1개 + 출처 1줄
3점: 슬라이드에 12줄 이상의 텍스트, 글자 크기 12pt, 그림 없음

## 3. 시각적 균형 (Visual Balance) — 20%

텍스트와 시각자료의 비율, 레이아웃 균형이 적절한가.

| 점수 | 기준 |
|------|------|
| 5 | 모든 슬라이드에 적절한 시각자료, 좌우/상하 균형 |
| 4 | 대부분 시각자료 있음, 일부 텍스트 치우침 |
| 3 | 시각자료 있으나 배치 불균형 |
| 2 | 대부분 텍스트만, 시각자료 부족 |
| 1 | 시각자료 전무, 텍스트만으로 구성 |

**자동 채점 기준 (`quality_scorer.py`):**
- 이미지/테이블 0개 + 텍스트 shape 3개 초과: 시각자료 부족 → 2점
- 이미지/테이블 1개 이상: 기본 충족 → 5점
- 그 외: 중간 → 3점

**권장 비율:**
- 텍스트 : 시각자료 = 40:60 ~ 50:50
- 듀얼 패널 레이아웃: 좌측(텍스트) + 우측(그림/차트)

---

## 4. 논리적 흐름 (Logical Flow) — 20%

슬라이드 간 전개가 자연스럽고, 전체 스토리라인이 일관적인가.

| 점수 | 기준 |
|------|------|
| 5 | 배경→방법→결과→결론 순서 명확, 전환 자연스러움 |
| 4 | 전체 흐름 양호, 1-2곳 전환 어색 |
| 3 | 큰 구조는 있으나 세부 전환 미흡 |
| 2 | 슬라이드 순서 혼란, 논리적 연결 약함 |
| 1 | 구조 없음, 무작위 나열 |

**수동 평가만 가능 (자동 불가)**

**체크리스트:**
- [ ] 전체 구조가 명확한가? (배경→방법→결과→결론 또는 STAR 등)
- [ ] 각 슬라이드의 핵심 메시지가 1문장으로 요약 가능한가?
- [ ] 이전 슬라이드에서 다음 슬라이드로의 전환이 자연스러운가?
- [ ] 결론이 서론에서 제기한 질문에 답하는가?
- [ ] 불필요한 중복 슬라이드가 없는가?

**학술 발표 권장 구조:**

```
1. Title Slide
2. Background / Problem Statement
3. Related Work / Current Limitations
4. Proposed Method / Architecture
5. Experiments / Setup
6-8. Results (주요 결과 → 세부 분석 → 비교)
9. Discussion / Limitations
10. Conclusion / Future Work
11. References
```

## 5. 형식 일관성 (Format Consistency) — 15%

폰트, 색상, 용어, 레이아웃이 전체적으로 통일되어 있는가.

| 점수 | 기준 |
|------|------|
| 5 | 폰트/색상/용어 완전 통일, 오탈자 없음, 정렬 일관 |
| 4 | 대부분 일관, 사소한 불일치 1-2건 |
| 3 | 눈에 띄는 불일치 3-5건 |
| 2 | 다수 불일치, 폰트/색상 혼재 |
| 1 | 통일성 없음 |

**자동 채점 기준 (`quality_scorer.py`):**
- 폰트 크기 종류 ≤ 7: 일관 → 5점
- 폰트 크기 종류 8-10: 약간 과다 → 4점
- 폰트 크기 종류 > 10: 과다 → 3점
- 최대 폰트 > 36pt: 경고

**체크리스트:**
- [ ] 본문 폰트가 전체 통일인가? (Pretendard 또는 Noto Sans KR 권장)
- [ ] 색상이 AMC 팔레트에서 벗어나지 않는가?
- [ ] 동일 개념에 동일 용어를 사용하는가? (예: "정확도" vs "accuracy" 혼용 금지)
- [ ] 줄임말 첫 등장 시 풀네임이 있는가?
- [ ] 숫자 표기가 통일인가? (소수점 자릿수, % 표기 등)

## 보고서 출력 예시

```
==================================================
  슬라이드 품질 보고서
  파일: presentation.pptx
  슬라이드: 11장 (11.693 x 8.267)
==================================================

  자동 점수 (자동측정 항목): 4.1/5.0

  [학술 근거] 수동 평가 필요 (가중치 25%)
    □ 모든 수치에 출처가 있는가?
    □ DOI가 유효한가?
    □ 가이드라인 참조에 판본이 명시되어 있는가?
    ℹ 수동 평가 필요: 핵심 주장에 DOI 레퍼런스가 있는지 확인

  [콘텐츠 밀도] 4.2/5 (가중치 20%)
    ✗ 슬라이드 1: 텍스트 부족 (32자)
    ✗ 슬라이드 8: 텍스트 과다 (2340자)

  [시각적 균형] 3.8/5 (가중치 20%)
    ✗ 슬라이드 4: 시각자료 없음 (텍스트만 5개)
    ✗ 슬라이드 6: 시각자료 없음 (텍스트만 4개)

  [논리적 흐름] 수동 평가 필요 (가중치 20%)
    □ 배경→방법→결과→결론 순서가 논리적인가?
    □ 각 슬라이드의 핵심 메시지가 명확한가?
    □ 전환이 자연스러운가?
    ℹ 수동 평가 필요: 슬라이드 간 전개가 자연스러운지 확인

  [형식 일관성] 4.5/5 (가중치 15%)
    ✗ 최대 폰트 크기 40pt (30pt 이하 권장)

==================================================

--- 우선 개선 항목 (높음→낮음) ---
1. 슬라이드 4, 6에 시각자료(차트/모식도) 추가
2. 슬라이드 8 텍스트 간소화 (핵심 포인트만)
3. 학술 근거 수동 검토 수행
```

## 점수 해석 가이드

| 총점 | 등급 | 해석 |
|------|------|------|
| 4.5+ | A | 발표 준비 완료 |
| 4.0-4.4 | B | 소소한 개선 후 발표 가능 |
| 3.5-3.9 | C | 주요 개선 필요 (1-2 차원) |
| 3.0-3.4 | D | 상당한 수정 필요 |
| < 3.0 | F | 전면 재작업 권장 |
